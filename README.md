# Nuclear Explosion Visualization

A stylized p5.js animation experiment that explores particles, light, color, and motion through an explosive visual effect.

## Tech stack

- JavaScript
- p5.js
- HTML Canvas

## Run locally

Open `index.html` in a browser. The repository includes its p5 runtime and does not require a build step.

## Status

Legacy creative-coding project from 2021. It is preserved as an early graphics experiment; the name describes the visual theme, not a scientific or predictive simulation.
